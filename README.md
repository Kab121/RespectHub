# RespectHub

RespectHub is a civic engagement platform that encourages residents to contribute positively to their communities.

Users can submit community actions such as volunteering, helping neighbours, or participating in local events. Administrators review submissions and award points that contribute to badges and leaderboard rankings.

## Features

- User registration and login
- Submit community actions
- Upload proof (photo/video)
- Admin review queue
- Points system
- Badge progression
- Leaderboard
